System Information Utility

A portable tool to quickly view system specs from your browser.

How to Use as a Portable Tool

Save this webpage (Ctrl+S or Cmd+S) as an HTML file onto a USB drive. You can then open this file in a web browser on any computer to view its specs, no internet connection required.

Inspiration Moodboard

This tool is inspired by the clean, data-dense interfaces of classic system utilities and modern dashboards. The goal is clarity, readability, and quick access to information.

Classic Utilities (Speccy, CPU-Z)

Inspiration from the no-nonsense, tree-structured layout that presents deep technical data clearly. The focus is on density and accuracy.

Modern OS Dashboards (Task Manager)

Modern OS tools use clean graphs and separated modules. We take inspiration from the clear categorization and use of whitespace.

Web Analytics Dashboards

Inspiration from web platforms like Vercel or Netlify. They excel at presenting data in clean, responsive "cards" with strong typography.
